IPL Mega Analytics Dashboard

🏏 An interactive IPL analytics dashboard built using Python, Pandas, and Power BI to analyze player performance, toss impact, team consistency, and venue-based insights across multiple IPL seasons.

Project Overview

This project focuses on transforming raw IPL match and ball-by-ball datasets into meaningful cricket analytics dashboards. The dashboard provides insights into batting performance, bowling performance, toss impact, and team consistency using interactive Power BI visualizations.
